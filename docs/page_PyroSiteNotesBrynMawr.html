1. Pyro Site Notes Bryn Mawr

Note that if you want to interactively stay in Python after running a script, use the "-i" flag: 

python -i NNand.py
(runs script)
>>> n.display()

Press <Control+d> to exit from Linux or Mac, <Control+z> to exit from Windows.
