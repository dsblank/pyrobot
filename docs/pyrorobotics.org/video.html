Pyro Tutorial Movies

The following Flash movies demonstrate various aspects of the Pyro Robotics project. To run them you will need a Shockwave Flash viewer, version 7 or greater.
  1. Getting started with Pyro: demonstration of starting Pyro and running PyrobotSimulator. 5 minutes and 50 seconds.
  2. Using the command line of Pyro's graphical user interface: demonstration of Pyro's Python interpreter with Stage's sonar. 11 minutes and 2 seconds.
If you have any trouble running these movies, or with running the examples demonstrated, please join our Pyro-users mailing list and let us know. Thanks! If you would like to know more about Pyro, please see PyroRobotics.org. If you would like to know more about how these movies are made, please see PyroHowtoVideo.
