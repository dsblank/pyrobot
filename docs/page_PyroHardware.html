Pyro Hardware

Pyro supports the following robotics platforms, and others including any Player-based robot:

  1. Using the Pioneer Robot
  2. Using the Hemisson Robot
  3. Using the Sony AIBO Robot
  4. Using the IntelliBrain-Bot
  5. Using the Roomba
  6. Interfacing Your Own Robot

Pyro supports the following simulators:

  1. The Pyrobot Simulator
  2. The Stage Simulator
  3. The Gazebo Simulator
  4. The Robocup Simulator

For more information and details on the simulations, see PyroModuleIntroduction especially the section on Robot Simulators.
